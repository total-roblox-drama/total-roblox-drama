## 🛠️ Common Issues & Fixes

### ❌ GUI Doesn't Appear After Execute
**Cause:** A conflicting overlay (Discord, GeForce Experience, Xbox Game Bar) is blocking the Rayfield render layer, or the script was executed before fully loading into the game.
**Fix:** Disable all third-party overlays, re-inject your executor, and execute the script again after you are fully spawned in the game server.

### ❌ Auto Obby Stops Midway Through Course
**Cause:** The obby layout was updated in a recent game patch and the pathfinding coordinates are no longer accurate.
**Fix:** Download the latest version of the script from the Releases tab. Auto Obby coordinates are recalibrated with every game update patch.

### ❌ Find Bag / Find Safety Statue Shows Wrong Location
**Cause:** The item spawn system in Total Roblox Drama uses randomized positions updated frequently. An outdated script version will point to stale coordinates.
**Fix:** Update to the latest release. If already on the latest version, toggle the feature off and on once to force a fresh scan of current spawn positions.

### ❌ Kill Aura / Fling Not Working in Current Round
**Cause:** The specific round type (e.g., Math Trivia, Voting phase) does not have eligible PvP targets, or the game is in a protected phase.
**Fix:** Kill Aura and fling only activate during appropriate combat/challenge phases. Wait until a PvP-eligible round begins and the feature will activate automatically.

### ❌ Script Not Loading on Android
**Cause:** The mobile executor does not support HTTP requests required for the Rayfield GUI asset fetch, or the executor version is outdated.
**Fix:** Update your mobile executor to the latest version. Ensure you have a stable internet connection during execution — the GUI requires one HTTP fetch on first load.

### ❌ "attempt to index nil" Error on Execute
**Cause:** The script was executed before the game's UI and character fully initialized.
**Fix:** Wait 5–10 seconds after fully spawning into the game before executing. If the error persists, it typically means the game was updated and a script patch is needed — check the releases tab.

### ❌ Auto Farm Collecting at Wrong Locations After Update
**Cause:** Game update moved item spawn zones or changed collectible names.
**Fix:** Download the latest patched version. The farm location map is updated with every game patch release.
